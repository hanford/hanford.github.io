facebook data visualizer
See detailed information about your facebook usage.

Step 1 - Download your data

1. To download your facebook data, click on this arrow which can be found on the top right of any page, than click on "Settings".

2. Once you are on the settings page, click "Download a copy of your facebook data" and then on the following page click on "Start My Archive".

3. After Starting your Archive, the waiting game begins. Usually within an hour facebook will email you notifying you that your archive is ready to be downloaded.

Step 2 - Grabbing the Application

Method 1 (easiest) - Download a zipped version of the application

Method 2 - Head over to the GitHub repo and clone then clone it.

Step 3 - Once you have the data and the parsing app

1. If you downloaded the parsing app unzip it.

2. Drag the file you downloaded from facebook named facebook-YOURNAME.zip into the main directory of the parsing application.

3. Rename the facebook-YOURNAME.zip file to just facebook.zip

Step 4 - Running the parser

Method 1 (easiest) - Right click "run-all.sh" and open the file in Terminal

Method 2 - unzip the facebook.zip file, run "npm install", then "bower install", after that you'll need to run "python -m SimpleHTTPServer" and then open a browser and go to localhost:8000

That's it!

From here you should have 8 interactive graphs using data that it totally unique to your facebook usuage!
