How I achieved the effect

This is something I was really confused about for a little bit, until it found clever solution. I am calling .header which has a higher z-index then .header-blur. Header-blur is placed directly under the header with the CSS filters that blur the contents when scrolling. This is kind of neat because it can provides some context to the navbar having content above it. It's subtle, but it's a great touch.

.nav {
position: absolute;
z-index: 3;
top: 0;
left: 0;
width: 100.5%;
height: 44px;
line-height: 44px;
padding: 0;
color: #eee;
font-size: 20px;
text-align: center;
}

.nav-blur {
position: absolute;
z-index: 2;
-webkit-filter: blur(3px);
-moz-filter: blur(3px);
-ms-filter: blur(3px);
-o-filter: blur(3px);
filter: blur(3px);
background: rgba(250,250,250,0.9);
top: 0;
left: 0;
width: 100%;
height: 44px;
padding: 0;
}

It's a little hacky on the actual webpage as I had to some unconventional things .. but it works great on actual websites :)


iOS 7 Knobs / Switches

These are some of the best things I have ever created in CSS. I am really proud of these because they resemble iOS 7 very well.. atleast in my opinion. They were also very challenging to create. I am using the HTML label tag to toggle the transparent checkbox. I first read about this functionality here on w3 suprisingly.

In my last job, our designer requested we use these as checkbox's and unforunately we didn't have enough time to implement these, because they really are pretty good looking if I say so myself..
